Interview Preparation

Sample Interview Q&A

These are representative questions students should be able to answer confidently after completing this course.

Architecture
What is the Medallion architecture and when would you use it?
A layered data organization pattern with Bronze (raw), Silver (cleansed), and Gold (business-ready) zones stored in ADLS Gen2. It is used when multiple teams need to consume data at different refinement levels, or when you need to preserve raw data for reprocessing and auditing.
ADF vs Databricks
How does Azure Data Factory differ from Databricks for data processing?
ADF is an orchestration and ELT service best suited for low-code data movement and pipeline scheduling. Databricks is a compute engine for complex, large-scale PySpark transformations. In practice, ADF is used to trigger and sequence Databricks notebooks within a larger workflow.
ETL / ELT
What is the difference between ETL and ELT, and which is preferred in Azure?
ETL transforms data before loading it into the target; ELT loads raw data first and transforms it inside the destination system. Azure architectures generally favour ELT because Synapse and Databricks provide scalable compute, and keeping raw data in ADLS Gen2 allows for full reprocessing when needed.
Databricks
What is Delta Lake and why is it preferred over plain Parquet?
Delta Lake adds ACID transactions, schema enforcement, time-travel, and MERGE support on top of Parquet files. It ensures data consistency in parallel write scenarios and allows engineers to roll back or reprocess from any historical snapshot when pipeline failures occur.
Data Warehousing
How do you implement Slowly Changing Dimensions (SCD Type 2) in Synapse?
Add effective_start_date, effective_end_date, and is_current columns to the dimension table. Use a MERGE statement in T-SQL to expire the existing record (update effective_end_date and set is_current = 0) and insert a new record with the updated attributes and is_current = 1 whenever a tracked attribute changes.
Real-Time
How would you design a real-time data pipeline for IoT sensor events?
Ingest events through Azure Event Hubs, process them using Databricks Structured Streaming or Azure Stream Analytics for windowed aggregations and anomaly detection, persist results to ADLS Gen2 or Synapse dedicated pools, and surface live insights via Power BI DirectQuery dashboards.
